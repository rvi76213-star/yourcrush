ADMIN COMMAND: !delete user

Description: Remove user from database

Usage: !delete user @username [reason] [permanent]

Parameters:
  @username - User mention (required)
  reason - Optional reason for deletion
  permanent - true/false (default: false)

Examples:
  !delete user @Spammer
  !delete user @InactiveUser "Inactive for 90 days"
  !delete user @Violator "Rule violation" true

Notes:
  ⚠️ WARNING: This action cannot be undone!
  - User must be mentioned with @
  - If permanent=true, all user data is deleted
  - If permanent=false, user is soft-deleted (recoverable)
  - Always provide reason for audit log
